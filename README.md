# RiffyCreates Dashboard

Static content dashboard for planning TikTok and Instagram posts under the RiffyCreates brand.

## What it includes

- Content idea bank with filters
- Weekly scheduling board with drag and drop
- Performance snapshot cards
- Trend alerts, batching prompts, CTA library, hashtag buckets, and inspiration feed
- Local browser persistence with import/export JSON

## Run locally

Serve the folder with any static server. For example:

```bash
/Users/rifat/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 -m http.server 4173
```

Then open [http://localhost:4173](http://localhost:4173).

## Deploy to Vercel

This project is static, so it can be deployed as-is on Vercel by importing the folder or pushing it to a Git repository connected to Vercel.
